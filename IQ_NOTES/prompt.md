## Reusable Prompt

> **"Create a note for `<concept>`"**

Replace `<concept>` with any JS/Playwright topic you want explained.

### Examples

| You say | I create |
|---------|----------|
| "Create a note for closures" | `IQ_NOTES\closures.md` |
| "Create a note for let vs var" | `IQ_NOTES\let_vs_var.md` |
| "Create a note for arrow functions" | `IQ_NOTES\arrow_functions.md` |
| "Create a note for promises" | `IQ_NOTES\promises.md` |

Each note will include code examples, explanations, and key takeaways — matching the style of `01_JS_Basics.md`.
